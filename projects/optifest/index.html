Pupil distance and some other calculations

Main functionality of this application is to calculate the distance between eye pupils. And couple more around the eyes.

Opticians use these calculations to cut glass fitting exactly to customer's eyes.

Traditionally simple rulers are used for these calculations. For example;

The aim was to produce easier, more robust, more accurate calculations using an android tablet. Which we accomplished.
